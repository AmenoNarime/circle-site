# 音響学における数学的基礎

音響学を理解するためには、数学的な基礎知識が不可欠です。この記事では、音響学で重要な数学的概念について解説します。

## 波動方程式

音波は波動現象の一種であり、以下の波動方程式で表現されます：

$$\frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u$$

ここで、$u$は変位、$c$は音速、$\nabla^2$はラプラシアンです。

## 正弦波の表現

最も基本的な音波は正弦波で表現され、次の式で記述されます：

$$y(x,t) = A \sin(kx - \omega t + \phi)$$

各パラメータの意味：
- $A$：振幅
- $k = \frac{2\pi}{\lambda}$：波数（$\lambda$は波長）
- $\omega = 2\pi f$：角周波数（$f$は周波数）
- $\phi$：位相

## フーリエ変換

複雑な音響信号を周波数成分に分解するために、フーリエ変換を使用します：

$$F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-i\omega t} dt$$

逆フーリエ変換は：

$$f(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} F(\omega) e^{i\omega t} d\omega$$

## 音圧レベル

音圧レベル（SPL）はデシベル（dB）で表現され、以下の式で計算されます：

$$\text{SPL} = 20 \log_{10} \left(\frac{p}{p_0}\right) \text{ dB}$$

ここで、$p$は測定音圧、$p_0 = 20 \times 10^{-6}$ Paは基準音圧です。

## 共鳴周波数

弦楽器の共鳴周波数は、弦の長さ$L$、張力$T$、線密度$\mu$によって決まります：

$$f_n = \frac{n}{2L} \sqrt{\frac{T}{\mu}}$$

ここで、$n = 1, 2, 3, ...$は倍音の次数です。

## オイラーの公式

複素数を用いた波の表現では、オイラーの公式が重要な役割を果たします：

$$e^{i\theta} = \cos\theta + i\sin\theta$$

これにより、正弦波を複素指数関数で表現できます：

$$y(t) = A e^{i(\omega t + \phi)} = A[\cos(\omega t + \phi) + i\sin(\omega t + \phi)]$$

## まとめ

これらの数学的概念は、音響学の理論的基礎を形成しています。実際の音響システムの設計や解析において、これらの式を適用することで、より深い理解と精密な制御が可能になります。

次回は、これらの理論を実際の楽器解析にどのように応用するかについて詳しく説明します。
